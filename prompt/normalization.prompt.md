# Objective
정규화 되지 않은 책 제목을 입력 받아 노이즈를 제거하여 **제목**, **부제목**, **회차**를 구조화 하여 추출 합니다. 

# Background Knowledge
입력 되는 책은 서브컬쳐 문화 계열의 책으로 종종 길고 설명적인 문장 형태로 구성됩니다.

## 노이즈
다음과 같은 단어/문장들은 **노이즈로**로 구분 합니다.
- 콘텐츠의 형태나 유형 내용등을 설명하는 단어나 문장 예:
  - "종이책", "전자책" 등의 발매 형태
  - "코믹", "소설" 등의 콘텐츠 유형
  - "캐릭터북", "설정집", "일러스트집", "단편집", "작품집", "모음집" 등 콘텐츠 내용
  - "개정판", "리커버" 등 변경이나 보완에 대한 내용
  - "완결" 등의 콘텐츠 상태
- 작품 외적인 요인을 나타내는 단어나 문장 (예: "애니화 결정", "동시 발매" 등)

# Instructions
- 가장 핵심적인 본문만 제목으로 판별하세요.
- 구분 기호(예: 콜론(:), 하이픈(-), 틸드(~) 등) 뒤에 있거나 구분 기호로 묵여 있는 단어나 문장은 부제목으로 판별하세요.
- 분기, 외전, 등장인물 등을 구체적으로 설명하는 단어나 문장은 부제목으로 판별하세요. (예: "Episode", "편", "Part", "Side", "Arc", "If")
- 회차는 다음 규칙에 따라 판별하며 숫자 외 단위는 제거합니다.
  - "3권" -> "3"
  - "1 ~ 7권 합본" -> "1~7"
  - "상편", "하편", "중편" -> "상", "중", "하"
- "제X장Y -Z"와 같은 특수 패턴은 아래와 같이 처리 합니다.
  - "제X장"은 큰 단위 구성 -> 부제목의 일부
  - "Y"는 그 장의 회차 -> 회차 필드로 분리
  - "Z"는 그 회차의 타이틀 제목 -> 노이즈로 제거
- 노이즈, 부제목, 회차로 추정 되더라도 제목과 연결하여 문맥상 의미를 구성한다면 제목으로 판별하세요.

# Step
1. 입력 받은 문자을 읽고 노이즈와 노이즈가 아닌 것을 구분 합니다.
2. 노이즈로 구분된 문장이나 단어를 제거 합니다.
3. 노이즈가 제거된 문장에서 제목과 부제목을 분리합니다. 부제목이 없을 경우 부제목은 null로 처리하세요.
4. 회차를 추출하세요. 회차가 없을 경우 회차는 null로 처리하세요.
